## Knowledge Gap Warning

If the project's pinned Next.js version is newer than the LLM's knowledge
cutoff (January 2026), the model will not know about APIs/conventions
introduced after that date. Always cross-reference this directory before
suggesting Next.js API calls when the version exceeds the cutoff.

`/setup-stack` populates this directory by fetching official Next.js docs
when the chosen version is post-cutoff.
