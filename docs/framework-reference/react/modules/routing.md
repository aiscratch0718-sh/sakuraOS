## Quick reminders

- Choose **one** router and stay with it (do not mix React Router and
  TanStack Router in the same app)
- Code-split routes with `React.lazy()` + `<Suspense>`
- Co-locate route data fetching with the route definition (loader pattern)
- Use protected route wrappers for auth-gated screens
